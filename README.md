# Age Calculator 🎂

A beautiful and responsive web-based Age Calculator built using **HTML, CSS, and JavaScript**.

This application calculates a user's exact age in **years, months, and days** based on the selected date of birth.

---

## 🚀 Features

* Modern and responsive UI
* Gradient background design
* Accurate age calculation (Years, Months, Days)
* Input validation
* Clean and simple user experience

---

## 📁 Project Structure

```
Age-Calculator/
│
├── index.html    # Main HTML file
├── style.css     # Styling file
└── script.js     # JavaScript logic
```

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

---

## 📌 How to Run the Project

1. Download or clone the repository.
2. Make sure all three files are in the same folder.
3. Open `index.html` in your browser.
